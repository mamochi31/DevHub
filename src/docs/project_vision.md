# 1. プロジェクト・ビジョン

**「自己学習のすべてを一つの場所で」**
> 学習フローの可視化、タスク管理、成果物の一元管理、そしてAIを活用した知識の定着までを完結させる、エンジニアのためのパーソナル・プラットフォーム。

## 2. 実装予定機能 (Backlog)

### 核心機能 (Core)

- **Learning Flow**: React Flow による学習ロードマップのフローチャート管理。

- **Portfolio Hub**: GitHub, Qiita, note 等の活動情報を集約。

- **Tech Blog**: Markdownベースの自律型ブログエンジン。

- **Book Library (New!)**:
  - 技術書の管理、ISBN連携による自動入力。
  - 学習フローとの紐付け（どの本でどの技術を学んだか）。
  - 読書メモからブログ記事への変換機能。

### 追加機能案 (Advanced Ideas) 🚀

- **Skill Radar Chart**: 技術スタックのバランスを可視化。

- **Learning Heatmap**: 日々の学習記録を GitHub 形式の草として表示。

- **AI Study Assistant**: 記事・書籍の要約、学習内容に基づいた確認クイズ生成。

- **Knowledge Linkage**: フローチャート、ブログ、書籍の三点を相互リンク。
